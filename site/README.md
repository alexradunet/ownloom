# Ownloom presentation site

Static presentation website for the Ownloom brand concept.

## Run locally

From the repository root:

```bash
python3 -m http.server 8080 --directory site
```

Then open <http://127.0.0.1:8080>.

## Files

- `index.html` — page content and structure
- `styles.css` — responsive visual design and loom animation
- `script.js` — small status-text rotation and reveal animation
- `assets/ownloom-mark.svg` — logo/favicon mark
